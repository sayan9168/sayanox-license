# How to Apply the Sayanox License

To apply the Sayanox License to your work, please follow these steps:

1. **Include the License File:** 
   Copy the `LICENSE` file from this repository and place it in the root directory of your project.

2. **Add Copyright Notice:**
   Attach the following boilerplate notice to your `README.md` or the main landing page of your project:
```text
   Copyright (c) [Year] [Your Name/Project Name].
   Licensed under the Sayanox License, Version 1.1. 
   See the LICENSE file for details.
```

3. **Source Code Headers:**
   For every source code file, add the header template found in `HEADER.txt` at the top of the file to clearly state the licensing terms.

4. **Attribution:**
   If you modify this license for your own use, you must rename it to avoid confusion with the official Sayanox License and clearly state that it is a modified version.
